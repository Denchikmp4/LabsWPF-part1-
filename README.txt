Лабораторные работы №1-3 по C# / WPF. Вариант 2.

Состав архива:
1. Lab1_Apps — пять отдельных WPF-приложений для лабораторной работы №1.
2. Lab2_SQLiteStudents_Variant2 — WPF-приложение SQLite. Дополнительное задание варианта 2: хранение и редактирование телефона студента.
3. Lab3_NetworkChat_Variant2 — клиент-серверный чат. Дополнительное задание варианта 2: отправка сообщения всем адресатам сразу.

Проекты рассчитаны на Visual Studio 2022 и .NET 8. Для ЛР2 пакет System.Data.SQLite.Core подключается через NuGet при восстановлении зависимостей.
